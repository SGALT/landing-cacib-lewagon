Python Level 1

Discover the program

Duration : 1 Day

Module 1 : Python 101

Objective: Understand why Python is such a great alternative to Excel and start programming. Notions and skills covered Types and variables, control flow String operations (concatenation, key methods) Conditions, Loops Workshops: Do several guided exercises using Jupyter Notebook to get familiar with the Python syntax and basic operations.

Module 2: Data Manipulation with Python

Objective: Discover the Python environment and key tools used for data manipulation. Run a first data analysis and plot your findings. Notions and skills covered Intro to Notebooks, Pandas, GitLab Basic data frame and array operations with Pandas and NumPy Basic data viz with Matplotlib and Seaborn (line, scatter plots) Workshops: Explore step-by-step simple datasets using Pandas in a Jupyter Notebook, create and play around a data frame.

Python Level 2

Discover the program

Duration : 2 Days

Day 1

Module 1: Python Refresher

Objective: Review Python basics and get ready for more advanced data manipulation challenges. Notions and skills covered : Types and variables, control flow String operations (concatenation, key methods) Conditions, Loops Workshops: Do several guided exercises using Jupyter Notebook to get familiar with the Python syntax and basic operations.

Module 2: Advanced Data Manipulation

Objective: Go deeper into data analytics and apply Python-based analytics methods to large financial dataset. Notions and skills covered Data pre-processing (key methods) Advanced data frame operations Data exploration and advanced visualization Workshops: Analyze large oil & gas dataset with Pandas, check data quality (n/a, nulls, duplicates), work on data format and run basic data exploration analysis (Boolean indexing, sorting, grouping, plotting).

Day 2

Module 1 : Data Visualization and Numpy 360

Objective: Get a good grip on Numpy and learn the data visualization best practices to communicate powerful insights Notions and skills covered : Matplotlib, Seaborn, Numpy Different types of chart and their use cases Statistical analysis with Numpy Workshops: Using the basic commands of Numpy, plot beautiful graphs using Matplotlib and Seaborn.

Module 2: Project

Objective: Run an end to end analysis on a retail sales dataset, from data collection to data visualization and learn how to efficiently communicate data insights. Notions and skills covered Data loading Data exploration Data cleaning Data enrichment Data analysis Data visualization
